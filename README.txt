# just-space

Файловая структура:
Перечисленные ниже файлы подключаются на странице в порядке нумерации.
1)Все константы выносятся в файл /include/const.php.
2)Все функции и сторонние модули выносятся в файл /include/init.php.
3)Задаем все мета-данные в файле /include/meta.php
4)Header сайта находится в файле /include/header.php
5)Footer сайта находится в файле /include/footer.php

Работа со скриптами:
1)Основной файл скриптов расположен в /js/script.js
2)При добавлении новых функций документируем их комментариями в которых указываем:
	а)Основное назначение функции
	б)Список входных параметров с описанием
	в)Список выходных параметров с описанием

Работа со стилями:
1)Основной файл стилей расположен в /css/style.css
2)По возможности добавляем только классы, id используется только для js.
3)При добавлении новых стилей заносим их в блок относящийся к определенному функционалу(например если стиль относится к блогу, то вносим его в блок обрамленный /* БЛОГ */ ... /* КОНЕЦ БЛОГ */)
